# Juice.com

Welcome to the Juice.com repository! This project is a dynamic web application built using React and Vite.

## Features
- Modern UI using React
- Fast development experience with Vite
- Responsive and performant

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/shubhamwadinkar2207-bit/Juice.com.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Juice.com
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack
- React
- Vite
- JavaScript / HTML / CSS
